┌──────────────────────────────────────────────────────────────────┐
│ 1️⃣  VERIFICAR CONEXIÓN (Siempre ejecuta esto primero)            │
└──────────────────────────────────────────────────────────────────┘

    npm run db:test

    ✅ Si ves "Conexión exitosa" → Continúa al paso 2
    ❌ Si hay error → Verifica tu archivo .env


┌──────────────────────────────────────────────────────────────────┐
│ 2️⃣  EJECUTAR SEEDER (Elige UNA opción)                           │
└──────────────────────────────────────────────────────────────────┘

    🔹 Para desarrollo/pruebas (15 min, 151 Pokemon):
       npm run seed:complete:gen1

    🔹 Para testing completo (45 min, 386 Pokemon):
       npm run seed:complete

    🔹 Para producción (2-4 horas, 1025+ Pokemon):
       npm run seed:complete:full


┌──────────────────────────────────────────────────────────────────┐
│ 3️⃣  VERIFICAR RESULTADOS                                          │
└──────────────────────────────────────────────────────────────────┘

    npm run query:stats

    📊 Verás las estadísticas de tu base de datos


┌──────────────────────────────────────────────────────────────────┐
│ 4️⃣  CONSULTAR DATOS                                               │
└──────────────────────────────────────────────────────────────────┘

    Ver un Pokemon específico:
    npm run query pokemon 25

    Buscar por nombre:
    npm run query search pikachu

    Ver tipos:
    npm run query types

╔══════════════════════════════════════════════════════════════════╗
║                    TROUBLESHOOTING                               ║
╚══════════════════════════════════════════════════════════════════╝

❌ Error: "Connection refused"
   → Ejecuta: npm run db:test
   → Verifica: archivo .env
   → Verifica: Security Groups en AWS

❌ Error: "Rate limited (429)"
   → No te preocupes, el seeder espera automáticamente

❌ Quiero limpiar la base de datos
   → Ver: SEEDER_GUIDE.md (sección Troubleshooting)
